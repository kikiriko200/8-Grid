8Grid

8Gridとは

8Gridは、グリッドレイアウトで12等分してもそんなに細分化しないやろw
という考えのもと、8つにしたらどうだろう…ということで作ったものです。
PC/タブレット/スマホの三種類のクラスを持っており、各対応ごとにCSSファイルを分けることもできます。

使い方

このグリッドレイアウトでは画面が8等分されます。
そのため、各カラムの合計値は常に8にしてください。
また、.v960を指定すると、このように960px幅を100%とします。

また、以上で紹介したクラスは複合して用いることで、より高度な表現が可能になります。
以下に例を示します。

PC/タブレット時のみ両端にサイドバーを置くタイプの3カラムレイアウト

.col-2,.col-m-2
.col-4,.col-m-4
.col-2,.col-m-2

この場合、.col-s-を指定していないので、スマホからは1カラムに見えます。

等幅3カラムレイアウト

.col-8,.col-m-8
.col-8,.col-m-8
.col-8,.col-m-8

これは特殊な例です。見ての通り、.col-8を3つ指定して強引に3カラムにしています。
最初に書いた各カラムの合計値は常に8という原則に反しますが、追加のクラス書くのも面倒くさいので、これはこのまま使いましょう。
なお、どうしても気になる人は追加のCSSを書きましょう。以下に例を示します。 

			/*CSS*/
.col-1-3{
	width:calc(100% / 3);
}
.col-2-3{
	width:calc(100% / 3 * 2);
}
