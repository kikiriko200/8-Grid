## Usage

You can use this like Bootstrap.

### class

1. for PC

```
.col-*
```

2. for Tablet

```
.col-m-
```

3. for Smartphone
```
.col-s-
```

4. 960px Grid(recommend to use to PC)
```
.v960
```

5. Trisect a row (v0.2~)
```
.col-1-3
.col-m-1-3
.col-s-1-3
```
